Freemarket SQL Interview

Overview

The SQL scripts in this repo contain the answers to the three sql questions (sent by Freemarket)

The dataset contains:

Account – account details and status

Client – client information, verticals, and group association

Group – GroupID and their GroupPod

InternalTransfers – transactions between accounts

DailyExchangeRate – currency conversion rates

Tasks:

1. Completed Transfers Between Different GroupPods (Q1 2025)

2. Identify transfers where sender and receiver are in different GroupPods during Q1 2025.

3. Total transfers for "Gambling" verticals in british pounds for 2024 and only live clients.

Notes:

Only completed transfers are considered (all queries)

Currency conversion was applied (to gbp)
